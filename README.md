# 🎯 Terminal Hangman Game

A professional terminal-based **Hangman Game** built with Python. The game allows players to guess randomly selected words from different categories while tracking high scores using JSON file handling.

---

## ✨ Features

- 🎮 Play Hangman Game
- 📂 Multiple Word Categories
- 🎲 Random Word Selection
- 🎨 ASCII Hangman Animation
- ❤️ Remaining Lives Counter
- 🔤 Letter Guess Validation
- 🏆 Top 5 Leaderboard
- 💾 JSON File Storage
- ⚠️ Exception Handling
- 🛡️ Input Validation

## 🎯 Game Flow

1. Enter your name.
2. Choose a word category.
3. Guess one letter at a time.
4. Every wrong guess decreases your remaining lives.
5. Guess the complete word before the Hangman is fully drawn.
6. Your remaining lives are saved as your score.
